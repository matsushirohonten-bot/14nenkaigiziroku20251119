🎯 重点的に伸ばすべき3つの分野

1. 自己主張スキルの向上

  • 現状：自分の意見を言うことが苦手、スタッフへの指示出しに躊躇
  • 改善策：アサーティブコミュニケーション研修の実施
  • 具体例：「私はこう思います」を主語にした伝え方の訓練
  • 効果：スタッフとの関係改善、組織運営の円滑化

2. 援助申し入れ能力の強化

  • 現状：困っても人に頼れない、一人で抱え込む
  • 背景分析：「怒られる」「断られる」「迷惑をかける」という不安
  • 改善策：チーム内で相談しやすい雰囲気づくり、定期的な面談
  • リーダーの気づき：松代さん自身が「攻撃性83点」と高く、それがスタッフの相談を阻害している可能性を認識

3. 問題解決行動能力（ロジカルシンキング）

  • 現状：問題が起きた時に論理的に解決する力が不足
  • 改善策：問題解決フレームワークの研修、ケーススタディ
  • 提案：AIツール（ChatGPT）の活用で思考の整理を支援
  • 成功事例：松代さんは5年前より問題解決能力が倍増（AI活用の効果）

🎯 重点施策

1. イベント開拓への転換

方針：訪問開拓からイベント開拓へ、プッシュ型営業からプル型営業へ

イベント場所の開拓先

  • 地域イベント：自治体HP、SNSから情報収集
  • 高齢者施設：サロン、デイサービス
  • 商業施設：スーパー、ショッピングモール
  • 医療機関：調剤薬局、クリニック
  • 企業：健康経営支援、福利厚生イベント
  • サービス業：集客コラボ（車ディーラー、保険会社など）
  • 郵便局：代理店（JPコミュニケーションズ）経由で全国展開可能

イベント営業のメリット

  • 訪問営業より接点のハードルが低い
  • 契約率が高い
  • 室内なら気候的リスクが低い（熱中症回避）
  • 即決が多く、サンプル使用率が高い
  • 回収の手間が省ける

デメリットと対応策

  • スペース費用がかかる → 代理店活用で軽減
  • 同じ場所で繰り返すと契約率低下 → 訪問営業も同様
  • スタッフの接客力が求められる → 配布回収型も選択肢

2. エッセンシャルワーカー開拓

  • ターゲット：介護職、医療従事者、学習塾講師など
  • 病院向け：セット販売
  • 学習塾向け：受験生・保護者向け説明会でのイベント

3. 休眠客掘り起こし企画

  • 休眠客向けチラシ（2種類作成中）
  • 既存客向け増本チラシ

4. ヘモグロビンA1cの積極展開

注意：利益率は宅配より低いため、積極推奨は控えめに

提案方法：「2週間に1セット（1ボール）」など新しい提案スタイル

位置づけ：宅配商品プラスアルファの売上確保
